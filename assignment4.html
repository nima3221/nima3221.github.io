Challenge 5: Accessibility Concerns

Since challenge 2 is structured using tables, any accessibility issues will stem from the formatting of the table.
Without proper structural markup, there won't be a way to differentiate between headersand data cells.
This can be especially difficult for people using screen readers to read the rows and columns out aloud.
For the table used in challenge 2, which uses one header, it's important to distinguish the content using "th" and "td" elements.
